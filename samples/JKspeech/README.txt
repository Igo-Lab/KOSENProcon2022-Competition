# Jomo Karuta speech database (JKspeech)
# 上毛かるた音声データベース

## 注意
- 本ファイルは「全国高等専門学校プログラミングコンテスト第33回群馬大会」の競技部門出場のために提供されるものです。
- 本競技部門実施を目的として使用許諾を得ています（許諾第03-05083号）。
- 目的外の利用および許可なく二次配布等を行うことを禁じます。

### Spec.
- Number of files : 88 (Japanese 44 + English 44)
- File format : WAVE (RIFF) format
- Sampling rate : 48 kHz
- Quantization : 16 bit, linear PCM
- Number of channels : 1 (monoral)
- Compression : No

### 仕様
- ファイル数 : 88 (日本語 44 + 英語 44) 
- ファイル形式 : WAVE (RIFF) 形式
- サンプリング周波数 : 48 kHz
- 量子化 : 16 bit , リニアPCM
- チャネル数 : 1 (モノラル)
- 圧縮 : なし

### 補足
- 日本語の音声ファイルは J01.wav ～ J44.wav
	+ ファイル名は「日本語読みにおける"あいうえお順"」を基準として連番を付与
- 日本語の読みは「上毛かるた平成27年版」を参照
	+ https://carta.media.gunma-u.ac.jp/gunma/jomo/index.html
- 英語の音声ファイルは E01.wav ～ E44.wav
	+ ファイル名は「日本語読みにおける"あいうえお順"」を基準として連番を付与
- 英語の読みは「Jomo Karuta. English Version.（群馬県立女子大学外国語教育研究所訳）」を参照
	+ https://carta.media.gunma-u.ac.jp/gunma/data_281jomoenglish/index.html
